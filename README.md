# AI Interviewer

**AI Interviewer** — это приложение для подготовки к собеседованиям с использованием искусственного интеллекта. Оно позволяет пользователям тренироваться, отвечая на типовые вопросы, получать обратную связь и улучшать навыки прохождения интервью.

Приложение помогает подготовиться к техническим и поведенческим вопросам, моделируя реальные собеседования и предоставляя персонализированные рекомендации.

## Стек технологий

- **Backend:** NestJS, JWT, PostgreSQL
- **Frontend:** React, TypeScript, Tailwind CSS
- **AI/ML:** OpenAI GPT API
- **DevOps:** Vercel, GitHub Actions

## Функционал

- Генерация вопросов для интервью с помощью AI
- Анализ ответов и обратная связь
- Сохранение истории интервью и прогресса пользователя
- Регистрация и авторизация пользователей
- Виджеты с заданиями
