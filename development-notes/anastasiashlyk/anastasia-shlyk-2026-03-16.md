# Дата: 2026-03-16

## Что было сделано

Сегодня я погрузилась в тесты для AI-чата. Сделала тесты как для фронтенда, так и для бэкенда. Для этого мы решили использовать Vitest, с которым я работала впервые. Сначала пришлось немного разбираться, как он вообще работает, но оказалось, что на самом деле ничего сложного — быстро разобралась.

Для тестов AgentPage сначала пришлось создать моковые данные вместо реального хука useChat.

В тестах проверяю, что:

- отображается приветственное сообщение ассистента;
- виден заголовок чата;
- ввод текста в поле вызывает setInput;
- кнопка Send вызывает send;
- а когда loading, поле и кнопка блокируются.

То есть UI реагирует на состояние и действия пользователя так, как задумано.

Для бекэенда я провела следующие тесты:

- проверка отправляет ли conversation-id в фронт энд
- проверяю правильно ли сохраняется сообщение пользователя
- проверка корректности стриминга
